# Cyber Project 1 Application

This project contains currently five flaws (including CSRF vulnerability) from OWASP top ten security risks for CSB Project 1.

## Installation

Requirements:
- Python 3.14
- Pip 25.3

Create and activate a virtual environment
```
python -m venv venv
```

```
Activate (Linux):
$ source venv/bin/activate

Activate (Windows):
$ venv\Scripts\activate
```

Install Dependencies
```
   pip install -r requirements.txt
```

##  Start the flask app

```
flask run
```
Open browser and go to http://localhost:5000
